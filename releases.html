v1.0

current Initial release

First public release of GioTux.

  • Boots straight into the ::GTX::> shell — filesystem/driver "loading" sequence, fullscreen takeover, and a custom color theme.
  • Full set of Unix-style file commands: ls, cd, cat, grep, find, nano, and more.
  • System tools — ps, kill, df, ipconfig, and a full-screen ginfo readout.
  • Admin mode with real system controls (services, firewall, disk repair, DISM, permissions) behind a confirmation step and an optional encrypted passcode.
  • Dedicated ADB / Fastboot console for Android devices.
  • A tiny .gtux app language (variables, input, conditionals) plus a game command for local batch games.
  • Companion Python GUI desktop, launched with gui.
  • cmd for a real, unmodified DOS-mode console with GioTux commands still available.
  • Permanent color customization via customize.
  • Assorted nonsense: cowsay, fortune, matrix, sudo, hack, and one command we're not listing here.
