What GioTux actually is

No installer runs a hidden virtual machine here. It's Command Prompt, reskinned and extended, still talking to the real Windows underneath.

Double-click GioTux.bat and it boots like a small OS: mounting a "filesystem," loading "drivers," dropping a custom theme on the window, then expanding to fill the screen so the illusion holds. What's left is a ::GTX::> prompt that behaves like a Unix shell — ls, cd, cat, grep, and friends all work the way you'd expect.

Underneath, it's still your machine. ps and kill talk to real processes, ipconfig reads your real network, and typing cmd drops you into an unmodified console session with GioTux's commands still layered on top. Type exit and it logs out cleanly, no trace left behind beyond a small color-preference file.

It also doesn't take itself too seriously — there's a fortune-cookie command, a cowsay, a fake "hacking" sequence, and at least one thing we're not going to explain here. You'll have to find it.

::GTX::> help

Available commands

Everything below runs from the same prompt. No menus to dig through — just type.

#

Shell & files

  • ls / list / dir browse the current folder
  • cd, pwd move around and check where you are
  • mkdir, rmdir create or remove directories
  • touch, rm, cp, mv the usual file operations
  • cat, grep, find read, search, and locate files
  • nano / vi a simple built-in text editor
#

System info

  • ps, kill list and stop running processes
  • whoami, hostname user and machine identity
  • uptime, date how long it's been running
  • df, ping, ipconfig disk space and network
  • ginfo full-screen system readout — OS, CPU, GPU, RAM, plus a live color test
#

Admin mode real controls

  • admin unlock system-level commands, confirmation required
  • services, firewall start/stop services, toggle the firewall
  • repair, checkdisk run sfc, DISM, and disk checks
  • setpass lock admin mode behind an encrypted passcode tied to your Windows account

Every destructive action asks before it runs. Nothing fires silently.

#

ADB / Fastboot mode

  • adb drop into a dedicated Android device console
  • install, push, pull manage apps and files on a connected device
  • fastboot ... flash partitions, read bootloader vars, reboot

Points at your existing platform-tools install — offers to fetch it if it can't find one.

#

Apps, games & GUI

  • run [app] execute a .gtux app — a tiny scripting language with variables, input, and if/goto
  • game [file] launch a batch game from a local games folder in its own window
  • gui open the companion Python desktop
#

DOS mode & theming

  • cmd switch to a real, unmodified console — GioTux commands still work alongside it
  • customize set your own background/text colors, saved permanently
  • customize default reset back to the classic look
#

Just for fun

  • cowsay a cow says whatever you type
  • fortune a random developer fortune cookie
  • matrix ambient rain in the terminal
  • sudo pretend to grant yourself power
  • hack [target] a harmless fake breach sequence
  • ??? there's a hidden command. Good luck.
